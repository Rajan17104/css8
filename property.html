Position property

Static

Static

element with static position can not move in any direction(top/left/right/bottom)

Relative

Relative

element with relative position can not move in all direction(top/left/right/bottom)

Absolute

Absolute

element with Absolute position can move according to body in all direction(top/left/right/bottom)

Absolute with parent

parent

element with absoluteposition can in parent.

Fixed

Fixed

element with fixed block set according to body.
not scroll with other element

sticky

sticky

relative + fixed
initially work as a relative.
whenever its position true (ex . top: 10px)work as a fixed.
